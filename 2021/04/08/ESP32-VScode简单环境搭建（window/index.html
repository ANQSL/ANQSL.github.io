ESP32+VScode简单环境搭建（window

准备

首先下载 git 2.21.0 版本：https://npm.taobao.org/mirrors/git-for-windows/
然后下载 ESP-IDF 工具安装器：https://dl.espressif.com/dl/esp-idf-tools-setup-2.3.exe

安装 git

安装页面弹出后一直点击 next
选择安装的目录

一直点击 next 直到安装

安装 ESP-IDF 工具安装器

1.同意这个协议
2.如果安装了 python 则选择安装的 python，如果没有安装则就安装 python3.7
3.选择刚刚安装 git 的路径
4.选择下载 esp-idf
5.选择 esp-idf 下载版本和下载的路径
6.选择 ESP-IDF Tools 安装的路径
7.然后一直点击 next 知道 install

vscode 配置

1.下载安装 espressif idf 工具
2.
3.自动选择安装的 python 路径

4 选择 ESP-IDF Tools 下载的 esp-idf

5. 6.点击下载
点击 get to next step
7.

最后

git 如果不想下载也可以同过 esp-idf tools 一起下载，只是速度较慢。其实也同过 vscode 的 espressif 工具一起下载，只是下载速度有点感人
